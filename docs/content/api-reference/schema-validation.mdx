## Overview

Argus captures and validates MCP tool schemas to help you understand and debug your tools.

## Schema Capture

When an MCP server is wrapped, Argus automatically captures:

- Tool definitions
- Input schemas
- Output types

## Viewing Schemas

In the dashboard, navigate to the **Schemas** tab to see all captured tool schemas.

## Schema Features

### Input Validation

See the expected input parameters for each tool:
- Parameter names
- Types
- Required vs optional
- Default values

### Response Structure

Understand the expected response format from each tool.

## Best Practices

### Consistent Schemas

Ensure your tools return consistent response structures to avoid confusion.

### Documentation

Use schema descriptions to document your tools clearly.

### Validation

Validate inputs before processing to provide helpful error messages.
